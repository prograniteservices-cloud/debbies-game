# Project Starry: The Nighttime Companion

A passive, voice-controlled "Ambient Companion" for kids that prioritizes sleep hygiene. Designed to transform the bedtime routine from a passive activity into an interactive, personalized odyssey that grows alongside the child.

## Tech Stack
- **Frontend / Core Logic:** Next.js (App Router), TailwindCSS, TypeScript
- **State Management:** Zustand
- **Backend / DB:** Supabase
- **AI Brain:** Gemini 2.5 Flash
- **AI Developer:** Gemini (Antigravity 2026)

## Philosophy
The application adheres strictly to the **Zero Authority** and **Anti-Expert** protocols, ensuring it serves as a companion rather than an educator or parent. The visual system features an atmospheric transition ("Good Night Mode") that limits blue light and adapts visually to support sleep.

See `gemini.md` for the global AI directives and `docs/schema.md` for data architecture.
