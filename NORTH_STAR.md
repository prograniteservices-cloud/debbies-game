# 🌟 NORTH STAR — Unicorn Island (Debbie's Game)

> **The single source of truth for this project. All agents read this first.**

---

## Mission

Create a beautiful, joyful, and highly engaging educational game for young children (ages 4–8). The game teaches **math** and **spelling** through play, using vibrant visuals, delightful sound effects, and personalized character profiles that make learning feel magical.

## Core Experience Pillars

1. **Delight First** — Every interaction should feel fun, not like schoolwork
2. **Progress Feels Real** — Visible level-ups, rewards, and celebrations at every milestone
3. **Sound Matters** — Rich, satisfying game sounds for every action
4. **Personalization** — Custom profiles, avatar selection, persistent progress
5. **Accessible** — Works on tablet/touch, mouse, and keyboard

## Difficulty Progression Rule

> **Difficulty increases every 5 levels, not 10.**
> Every 5th level milestone triggers a special **Popping Level** (pure fun/decompression).

```
Levels 1–5   → Tier 1 (easiest)
Levels 6–10  → Tier 2
Levels 11–15 → Tier 3
Levels 16–20 → Tier 4
Levels 21–25 → Tier 5
...and so on
```

After every 5 levels: 🎉 **POPPING LEVEL** — Pop as many items as possible before the timer runs out!

## Visual & Audio Bar

- Animations must be premium, fluid, and purposeful (use Anime.js + Framer Motion)
- Music: cheerful ambient background loop, adaptive to game state
- SFX: satisfying pops, sparkles, correct-answer chimes, wrong-answer boing, level-up fanfare
- Art style: vibrant flat/gradient with soft glow effects, glassmorphism cards
